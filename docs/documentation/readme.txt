!!! ВСЕ Абзацы текста НАДО оформлять в <p></p>, а не использовать <br />.

В документациях использовать только XHTML валидный код (это не <br>, а <br />, не <b>, а <strong>, не <i> а <em> и т.д.)

Списки делать через <dl> + <dd> + <dt> + <ul> + <li>, примеры как сделать есть (см. индекс или в разделе 3.1)

Для вставки кода из файла используется <!-- code цифра --> , эта строка будет заменена покрашенным кодом из файла codes/cat-цифра.php, пример в разделе 1.2
Для вставки кода runtime: <<code>>код<</code>> (пример в разделе 3.1)
Для примечаний: <<note>>примечание<</note>> (пример в разделе 2.2)
